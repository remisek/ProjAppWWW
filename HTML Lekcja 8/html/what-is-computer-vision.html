1. What is Computer Vision

Computer Vision (CV) is a field of Artificial Intelligence (AI) that uses Machine Learning and Neural Networks to teach computers and systems to derive meaningful information from digital images, videos, Point Clouds and other visual inputs.

Most popular sub-domains of Computer Vision:

- classification common use case: quality control in manufacturing
format: image -> class
architecture: CNN with MLP at the end that outputs probabilities vector of length equal to number of classes
loss: cross-entropy loss
example use cases: identify product (e-commerce), recognize facial expression, defect detection (quality control)

- detection
common use case: face detection in smartphones
format: image -> class & x,y coords
architecture: in basic approach CNN with 2 outputs: regression and classfication
loss: cross-entropy for class (classification) and MSE for bounding box coordinates (regression)
example use cases: tracking/counting people/items (monitoring), pose tracking

segmentation
common use case: self-driving car environment mapping
format: image -> image-like (segmentation mask)
architecture: UNet
loss: binary-cross-entropy for each pixel in simplest binary segmentation case, dice-loss for multi-class (model outputs tensor of size [width, height, num_classes] - for each pixel prob. of each class)
example use cases: detecting tumors, autonomous vehicles, background removal
